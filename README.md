# Dodge Bug

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT) 

A terminal-based Dodge Bug implementation written in JavaScript (Node.js).

# How to Play

```
yarn or npm install
yarn play or npm run play
```
